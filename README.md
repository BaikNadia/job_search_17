# Job Search 17

Консольное приложение для поиска вакансий через HeadHunter API с фильтрацией и сортировкой.

## 🚀 Возможности

- 🔍 Поиск вакансий через HeadHunter API
- 💰 Фильтрация по минимальной зарплате
- 🎯 Фильтрация по ключевым словам
- 📊 Сортировка по зарплате (по убыванию)
- 💾 Сохранение результатов в JSON файл
- 📝 Вывод топ-N вакансий

## 📦 Установка

### Требования
- Python 3.11+
- Poetry (для управления зависимостями)

### Установка зависимостей

\\\ash
# Клонирование репозитория
git clone https://github.com/BaikNadia/job_search_17.git
cd job_search_17

# Установка зависимостей через Poetry
poetry install

# Активация виртуального окружения
poetry shell
\\\

## 🎮 Использование

### Запуск приложения

\\\ash
python -m src.main
\\\

### Пример использования

1. Введите поисковый запрос (например: \"Python разработчик\")
2. Укажите количество вакансий для вывода (например: 10)
3. Введите ключевые слова для фильтрации (например: python django)
4. Укажите минимальную зарплату (например: 100000)
